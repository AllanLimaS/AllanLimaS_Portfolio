title: "FreelaFinder"
description: "Automated freelancer opportunity filter with AI, Telegram notifications, and Google Sheets logging. The system connects to Gmail via IMAP to read notification emails for new opportunities, extracts project data from the page via web scraping, and classifies each one with an LLM according to a profile of preferences and skills I described (fit score, red flags, proposal draft). Everything is logged in a Google Sheets spreadsheet and a notification is sent to Telegram with the score visually highlighted, projects with good fit already include a ready-to-copy proposal draft. This way I don't need to read through dozens of listings every day, I only see the ones that are already a good match for my profile."
